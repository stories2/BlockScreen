화면을 숨기고는 싶은데 그렇다고 핸드폰이 꺼져서는 안되는 작업들이 있다
이러한 부분들을 위해 화면만 가려주는 솔류션을 개발하려 한다
평상시에는 화면에 작은 아이콘이 떠 다니다가 사용자가 해당 아이콘을 누르게 되면 자동으로 화면을
가려주는 것이다
또한 다시 그 아이콘을 누르면 다시 화면이 원래데로 돌아온다
이를 위해서는 부팅시에 본 솔류션을 구동해야 할 것이며
전원키를 눌러서 화면을 끈 상태에서는 작동을 최소한으로 줄여야 할 것이다
네트워킹을 통해 사용자의 정보를 분석하는 매커니즘이 필요하고, 평상시에 떠 있는 아이콘이 
움직일 수 있게 해야 한다
또한 2개의 서비스를 두어서 한개는 터치가 인식되도록 하고, 한개는 터치가 인식되지 않게 해야한다
터치가 인식 되는 서비스는 사용자가 컨트롤 하는것에 대한 처리를 담당하고,
터치가 인식 되지 않는 서비스는 화면을 가리고, 보여주는 기능을 처리한다
오프라인처리도 필요하다, 그러기 위해서는 클라이언트 데이터베이스 개념을 구축 해 놓아야 한다
